fn find_wolfe<Func>(&self, c: S, f: Func, hint: Option<(S, S)>) -> Result<S, Secant2Error> where Func: FnMut(S) -> (S, S)

Find an approximate minimum of a function ϕ satisfying the Wolfe condition.

  • f should be the function f = |x| (ϕ(x), ϕ'(x)).
  • c specifies the initial search interval (0, c).
  • hint may contain the value (ϕ(0), ϕ'(0)) to avoid unnecessary evalution if this value is already known (as is the case of the NonlinearCG minimization method).
